# Voice Samples

Place your voice reference audio files in this directory to use them with the API.

## File Naming Convention

Name your audio files as `{voice_name}.wav` or `{voice_name}.mp3`

Example:
- `default.wav` - Use with `voice: "default"` in API requests
- `female.wav` - Use with `voice: "female"` in API requests
- `male.wav` - Use with `voice: "male"` in API requests

## Audio Requirements

- **Minimum duration**: 3 seconds
- **Format**: WAV or MP3
- **Quality**: Clear speech recording recommended
- **Content**: A sample of the target voice you want to clone

## Adding a Default Voice

To get started quickly, add a voice sample named `default.wav` in this directory.

You can use any clear speech recording (at least 3 seconds) as your default voice.
